Last Lesson : ミニアプリケーション

Last Lesson : ミニアプリケーション

最後に、今回学んだことを使って簡易的なeCommerce Shop(EC サイト)体験アプリケーションを作ってみましょう。

まずは内容を列挙して見るので、出来上がるものを想像してみましょう。

  1. pugのテンプレート機能を使って/、/about、/contactを用意しましょう。その時のルーティングも用意しましょう。
  2. /aboutと/contactのメニューに対して、訪れているページのリンクがアクティブになるように設定しましょう。
  3. /contactページにて、お問い合わせフォームを設置しましょう。
  4. /productを用意し、sample/mini-app-sample/products.jsonにある商品リストを表示しましょう。（商品画像はsample/mini-app-sample/sample.jpgを利用可）
  5. 購入ページ/orderを用意し、商品を購入できる様なフォームを設置しましょう

以下ではある程度の内容を項目に沿って提示しますが、復習のためにも、まずは自身の力だけでやってみるのも良い練習になるでしょう。

(* デザインにはBootstrapを適用していますが、本ハンズオンのメインではないためおまけ程度にとどめておいてください)

ベースとなる環境を構築

$ mkdir mini-app
$ cd mini-app
$ npm init

mini-app/index.jsを作成

var express = require('express');
var app = express();

app.get('/', function(req, res) {
  res.render('index');
});

app.listen(3300, function() {
  console.log('Starting mini application server at http://localhost:3300');
});

/aboutと/contactのルーティングを用意

var express = require('express');
var app = express();

app.set('views', './views');
app.set('view engine', 'pug');

app.get('/', function(req, res) {
  res.render('index');
});

app.get('/about', function(req, res) {
  res.render('about');
});

app.get('/contact', function(req, res) {
  res.render('contact');
});

app.listen(3300, function() {
  console.log('Starting mini application server at http://localhost:3300');
});

4. /productページを用意し、products.jsonにある商品リストを表示

public/images/sample.jpg、data/products.jsonを用意

index.jsに以下を追加

app.use('/assets', express.static(__dirname + '/public'));
var products = require('./data/products.json');
app.get('/product', function(req, res) {
  res.render('product', {
    title: 'Mini App - product',
    products: products,
    activeMainMenu: {
      product: 'active',
      about: '',
      contact: ''
    },
  });
});

product.pugを用意

              li Category : #{product.category}

5. 購入ページ/orderを用意し、商品を購入できる様なフォームを設置

index.jsに以下を追加 (※ 購入用postはデータを確認するだけにとどめておく)

app.get('/order', function(req, res) {
  console.log('req.query = ', req.query);
  var product = products.find(function(product) {
    return product.id === req.query.id;
  });
  console.log('show product = ', product);
  res.render('order', {
    title: 'Mini App - product / order',
    product: product,
    activeMainMenu: {
      product: '',
      about: '',
      contact: ''
    },
  });
});

app.post('/order/:id', function(req, res) {
  console.log(req.params);
  console.log(req.body);
  res.redirect('/thanks');
});

product.pugでのhref="#"を以下に変更

href="/order?id=" + product.id

order.pugを用意

extends layout.pug

注意

ここで提示するサンプルは、実際に運用するには足りない機能が多く残っています。

例えば、

  • 在庫確認機能(データに参照をかけて、購入可能かどうかのチェックが必要です)
  • 商品を管理するデータベース（代替としてjsonを使用しましたが、テキストでデータを扱うのはやめましょう）
  • メーラー（お問い合わせや商品購入結果用に必要ですね、また、セキュリティも必要でしょう）
  • 例外処理やBad request対応
  • スパムやBot対策 などなど、ユーザーの個人情報などを扱う際は特に気をつけたいですね。 まだまだやること、学ぶことは多いかもしれません。あとは、自分が何を作りたいのかによってコーディングする内容は変わってきますが、 みなさんのWebアプリケーション構築の手助けになれば幸いです。
