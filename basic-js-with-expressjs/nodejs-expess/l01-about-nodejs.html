Lesson 01 : Node.js

Lesson 01 : Node.js

Install

インストーラー

公式サイトからLTSバージョンをダウンロードし、インストール。

コマンドライン

OSX

Windows

インストール確認方法

コマンドライン上で、 node --version とタイプ。

インストールしたバージョンが表示されれば完了。

Windowsの場合

インストールしたnode専用コマンドプロンプトを使用することをお勧めします。

nodeコマンド

インストールすることで、 node コマンドが使用可能となりました。

node --version の --version はインストールされているバージョンを確認するためのオプションのひとつです。

nodeとだけ打つとnode.js（JavaScript）を対話形式で実行できます。(REPL : Read-Eval-Print-Loop)

$ node
> 1 + 1
2
> str = 'hello node.js'
'hello node.js'

node と打ってREPLを起動してから終了する場合は、Ctrl(コントロール)+Cを２回入力することで終了できます。 (* 一回Ctrl+Cを入力すると以下のようになるので、もう一度Ctrl+Cを押して終了してください)

>
(To exit, press ^C again or type .exit)
>

node [javascript file path] と打つことで、jsファイルを実行することも出来ます。

hello.js を任意の場所へ作成し、以下の様に記述。

console.log('hello node.js');

現在のディレクトリから作成したパスとファイル名を指定し以下のように実行すると hello node.js が出力されます。

( 現在のディレクトリにファイルを作成する場合は、パスの指定は省略できます。 node hello.js ) ( nodeと打ってREPLに入った状態ではないことに注意してください)

$ node somewhere/hello.js
'hello node.js'

練習

  1. node と打って、自由に試してみよう
  2. 適当な場所に適当なコードを書いたjsファイルを保存して、node [file name] で実行させてみよう
