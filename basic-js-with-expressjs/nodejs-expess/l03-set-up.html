Lesson 03 : アプリケーション構築開始

Lesson 03 : アプリケーション構築開始

自身のPCの好きな場所に今回作業するディレクトリを作成しましょう。

(コマンドは以下の様になります、ディレクトリ＝フォルダを作成さえできればどんな方法でも構いません)

[~]$ mkdir ebisujs

作成したディレクトリ内でnpmコマンドを使ってpackage.jsonを作りましょう。

（基本情報を求められますが、全てエンター（未入力）で大丈夫です）

[~/ebisujs]$ npm init

※メインとなるファイルはデフォルトではindex.jsとなります

これでNode.jsによるアプリケーション構築の為の最初の段階は終了です。簡単ですね。

それでは、いよいよWebアプリケーションを作って行きましょう。その前に、expressをインストールしておきましょう。

[~/ebisujs]$ npm install --save express

インストールできたら、index.jsをnpmコマンドを実行したディレクトリ(ここではebisujs)に作成・開き、以下のコードを記述しましょう。

var express = require('express');
var app = express();

app.get('/', function(req, res) {
  console.log('someone came!');
  res.send('Welcome my world!');
});

app.listen(3000, function() {
  console.log('Starting your application server at http://localhost:3000');
});

現在のディレクトリ構成は以下のようになっていると思います。

ebisujs
|- node_modules
|-package.json
|-index.js

書き終わったら保存し、以下のコマンドで実行させましょう(.jsは省略可)

[~/ebisujs]$ node index.js

実行すると、Starting your application server at http://localhost:3000という文字列が表示されたかと思います。

それでは、ブラウザのアドレスバーにhttp://localhost:3000と入力してブラウザに Welcome my world! と出力されることを確認してみましょう。

Express

Expressとは、npmパッケージの一つで、webアプリケーションフレームワークの一つです。

ようするに、Node.jsでwebアプリケーションを構築する上で必要となる機能をまとめてくれたパッケージの１つになります。（他にもExpressのようなwebアプリケーションフレームワークはありますが、ここでは一般的に使われるExpressをとりあげます。）

require('express')

Node.jsはいくつかの予約語があり、require関数はその一つになります。

require([package name])とすることで、npm installでインストールしたパッケージを使用することが出来ます。

自身で作成したjsファイルなどを読み込む場合もrequire([path/to/yourfile])として書いて使用することが出来ます。

その際は、

module.exports = function() {
  console.log('Original module');
};

として、エクスポートしてあげる必要があります。
