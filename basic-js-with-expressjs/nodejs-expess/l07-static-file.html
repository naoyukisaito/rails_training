Lesson 07 : 静的ファイル

Lesson 06までの内容で、基本的なWebサイトの構築は可能となりました。

あとは、フロント側（ブラウザー側）で表示する画像や適用するCSS, JavaScriptなどを提供できるように設定をしましょう。

以下のように設定することで、server内のindex.jsと同階層のpublicディレクトリ下を/staticルートとして公開することになります。

app.use('/static', express.static(__dirname + '/public'));
