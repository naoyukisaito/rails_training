Lesson 04 : Express基本解説

Lesson 04 : Express基本解説

Lesson 03で書いたコードをもう一度見てみましょう。

var express = require('express');
var app = express();

app.get('/', function(req, res) {
  console.log('someone came!');
  res.send('Welcome my world!');
});

app.listen(3000, function() {
  console.log('Starting your application server at http://localhost:3000');
});

Lesson 03の最後にhttp://localhost:3000へ訪れた際、node index.jsで起動したコンソールにsomeone came!と表示（出力）されていることに気が付きましたか？ブラウザを再読み込みしたりすると、アクセスする毎に出力されるかと思います。

express()を受け取ったappオブジェクトは、様々なプロパティをもち、app.get()はHTTP GETリクエストでアクセスした際の動作を設定する関数になります。もちろん、HTTP POST等も設定可能です。

app.get()の'/'は、urlのrootを意味します。

app.get()のfunction(req, res)のreqは、requestを意味し、アクセスがあった際の情報とメソッドが格納されています。

app.get()のfunction(req, res)のresは、responseを意味し、応答する際の情報とメソッドが格納されています。res.send()で必ず反応を返して上げる必要があります。

app.listen()は、サーバーの起動を開始する関数になります。

3000という数字は、ポート番号を意味し、コンピュータ上の他のサービスと被らない番号を指定してあげます。

expressを利用し、数行のコードを書いてwebアプリケーションの基礎となるものを作る事ができました。

JavaScriptの文法基礎

Object

var obj = {
  test1: 'this is string',
  test2: ['this', 'is', 'array'],
  test3: {
    str1: 'this',
    str2: 'is',
    str3: 'object',
  },
  test4: function() {
    console.log('this is function!');
  }
}

console.log(obj.test1); // 'this is string'
obj.test4(); // 'this is function!'

callback function

// コールバックファンクションを第二引数に持つtest関数の定義
function test(something, callbackFunc) {
  var example = 'I got ' + something;
  callbackFunc(example);
}

// test関数を実行
test('hoge', function(str) {
  console.log(str); // 'I got hoge'
});
