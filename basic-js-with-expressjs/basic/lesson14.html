Lesson 14 : 体験編 HTML + JavaScript

Lesson 14 : 体験編 HTML + JavaScript

体験として、学んだ内容を駆使したり、自分で調べたりしてJavaScriptでHTML操作をしてみましょう。

samples/appendex-lesson14/exercise_14.jsに処理を書き、samples/appendex-lesson14/exercise_14.htmlをブラウザで表示して正しく動作させてみましょう。

HTMLの操作等を助けてくれるjQueryライブラリを使ってみましょう。

https://jquery.com/

https://learn.jquery.com/

解答例

※以下は解答例になります。参考にしてみてください。

function getDayStr(dayNum) {
  days = ['日', '月', '火', '水', '木', '金', '土'];
  return days[dayNum];
}

jQuery(function(){
  $('#sample01 button').click(function(){
    console.log('1: 1週間の曜日をリストにしましょう');
    days = ['日', '月', '火', '水', '木', '金', '土'];
    days.forEach(function(e) {
      $('#sample01 ul').append('<li>' + e + '</li>');
    });
  });
  $('#sample02 button').click(function(){
    console.log('1: JavaScriptという文字列をカウントしてみましょう');
    textLines = $('#sample02').text();
    countString = textLines.match(/JavaScript/g).length;
    $('#sample02 button').after('<p>JavaScriptという文字列は、' + countString + '回出現しています。');
  });
  $('#sample03 button').click(function(){
    console.log('1: 今日の日付と時間を表示しましょう');
    today = new Date();
    $('.year').text(today.getFullYear());
    $('.month').text(today.getMonth());
    $('.date').text(today.getDate());
    $('.day').text(getDayStr(today.getDay()));
    $('.hour').text(today.getHours());
    $('.minutes').text(today.getMinutes());
    console.log('2: 東京の天気を取得して表示しましょう');
    /*
       天気を取得する為に、Freeで使えるAPIを使ってみましょう。
       今回はここのAPIを使うことにします。
       https://home.openweathermap.org/
       ドキュメント：http://openweathermap.org/current
       ユーザー登録をしてAPI Keyを取得してください(※下記のappidがapi keyです。一定期間は製作者のkeyを使える状態にしておきます。)
    */
    $('.weather').text('取得中');
    $.ajax({
      type: "GET",
      url: "http://api.openweathermap.org/data/2.5/weather?q=Tokyo&appid=eb20b4bc1d5d90c2e9c9c813f7632bd4",
      dataType: "json",
      success: function(result) {
        $('.weather').text(result.weather[0].main);
      }
    });
  });
});
