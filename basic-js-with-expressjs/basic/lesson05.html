Lesson 05 : 変数とは？

Lesson 05 : 変数とは？

さて、以前までの演習ではその場その場で値を出力してきました。

計算した値を保存して、使用したい時に呼び出すことができればやれることの幅が広がります。

プログラミングでは、その為の格納場所を任意に定義することができ、それを「変数」と呼びます。

JavaScriptでは、var hensuu = '変数';(hensuu = '変数';でも可)のように記述します。

変数には、プログラミングをより便利にするためにいくつかの種類が用意されています。

JavaScriptの変数をちょっとだけ体験してみましょう。

例）
var number_val = -432.32;
var string_val = '文字列\nです。';
var bool_val = false;

練習問題

var number_val = 123;
var string_val = '文字列\nです。';
var bool_val = true;

// 1. 各変数を自由に出力させてみましょう。

// 2. 各変数を自由に変更して出力してみましょう。

// 3. 四則演算式を書いて代入し、出力してみましょう(5 * 4など)

// 確認 : number_valは変数で数値なので、以下が正常に実行されます
// console.log(number_val + 100);// 100 が足されて出力されます
// number_val = number_val + number_val;
// console.log(number_val);// 変数同士の演算も可能です、変数同士の演算結果を変数に格納できます


// a += b は a = a + bを意味します
string_val += '\n文字連結には+を使います。';// \nは改行を意味します
console.log(string_val);

解答例

※以下は解答例になります。参考にしてみてください。

var number_val = 123;
var string_val = '文字列\nです。';
var bool_val = true;

// 各変数を自由に出力させてみましょう。
console.log(number_val);
console.log(string_val);
console.log(bool_val);

// 各変数を自由に変更して出力してみましょう。
number_val = 456;
console.log(number_val);
string_val = 'this is string.';
console.log(string_val);
bool_val = false;
console.log(bool_val);

// 四則演算式を書いて代入し、出力してみましょう(5 * 4など)
number_val = 2 + 5;
console.log(number_val);

// number_valは変数で数値なので、以下が正常に実行されます
console.log(number_val + 100);// 100 が足されて出力されます
number_val = number_val + number_val;
console.log(number_val);// 変数同士の演算も可能です、変数同士の演算結果を変数に格納できます


// a += b は a = a + bを意味します
string_val += '\n文字連結には+を使います。';// \nは改行を意味します
console.log(string_val);
