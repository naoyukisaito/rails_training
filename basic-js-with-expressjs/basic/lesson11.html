Lesson 11 : JavaScript国のルール

Lesson 11 : JavaScript国のルール

これまで、プログラミングに必要となる基本を学んできました。

その過程で、JavaScriptのルール（規約や文法）を体験してきたと思います。

変数の宣言時には、varをつける。1つの処理の終わりには;(セミコロン)をつける。

functionやfor、while、ifなどは{}で括る。Objectを表現する歳も{}を使う。配列は[]。などなど。

プログラミング言語毎にこれらの規約や文法は異なりますが、似ているものもあるので他言語での方法を調べてみるのも良いでしょう。

それでは、他にどのようなルールが存在するのでしょう。簡単に紹介します。

プログラミング言語を学ぶ上で、分からないことがあれば公式ドキュメントや信頼性のおけるサイトで調べることが基本となります。

書籍も頼りになりますが、情報が古かったりするので注意が必要です。不特定な人の情報も、注意が必要です。

例として、以下のサイトを提示しておきます。

https://developer.mozilla.org/ja/docs/Web/JavaScript

JavaScriptには、バージョンが存在します。現在一般的に使われている規格(対応されている規格)は、ES5となるのでご注意ください。ESとは、ECMAScriptの略で、標準化名。

ともあれ、JavaScript中級、上級への足がかりであったり、足りない知識は上記のサイトなどから手に入れましょう。

練習問題

// if文の書き方
if(true) {
  console.log('{}でくくりますが。');
}
if(true) console.log('1プロセスだけならば、{}を省略することができます。');

// 以下のfor文は無限ループになります。３箇所全て省略が可能です。
/*for(;;) {
}
*/

//while文の種類として、do while文もあります。do{}部分が最初に処理されてからwhileの条件内で{}部分が処理されます。
i = 0;
do{
  i++;
  console.log(i);
} while(i < 3)

// switch文 : 特定の変数が与えられた場合、決められた処理を行う
something = 'red';
switch(something) {
  case 'red' :
    console.log('赤');
    break;
  case 'green' :
    console.log('緑');
    break;
  case 'blue' :
    console.log('青');
    break;
  default:
    console.log('何色?');
    break;
}

// typeof演算子 : 変数が何者かを教えてくれる便利な演算子
var getHeight = function(){};
var shape = "round";
var size = 1;
var human = true;
var gene = [];
var book = {};

console.log(typeof getHeight);
console.log(typeof shape);
console.log(typeof size);
console.log(typeof human);
console.log(typeof gene);
console.log(typeof book);
