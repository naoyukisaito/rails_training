Lesson 03 : Boolean(bool型)とは？

Lesson 03 : Boolean(bool型)とは？

プログラミングにおいて、型というものが重要になってきます。

Lesson01, 02では、文字列(Stringg型)と数値(Number型)を学習しました。

Boolean(bool型)とは、true(真), false(偽)の2つを表現する型になります。

例)
console.log(true);
console.log(false);

比較演算子とは、2つの値を比べる為の記号（演算子）で、比較結果（演算結果）をtrueまたはfalseで返してくれます。

console.log(1 === 1);// -> true
console.log(1 === 2);// -> false

論理演算子は、日本語で表現すると以下のようになり、それぞれの記号で表現します。 「〜かつ〜」(AND) : &&

「〜または〜」(OR) : ||

「〜ではない」(NOT) : !

console.log(true && true);// -> true
console.log(true && false);// -> false
console.log(true || false);// -> true
console.log(!true);// -> false

練習問題

// 比較演算子の効果を体験してみましょう。
console.log(100 > 99);// true 「100は、99より大きい」は真(true)
console.log(100 <= 99);// false 「100は、99以下」は偽(false)
console.log(100 >= 100);// true 「100は、100以上」は真(true)

以下の実行結果がどうなるか想像した上で実行させてみましょう。

演算子（記号）には、処理される優先順位が決まっています。

また、丸括弧()で部分をくくると、全体の中でその中身がまず処理されます。

console.log('===は、厳密に等しい("===は、厳密に等しい" === "===は、厳密に等しい")');
console.log('===は、厳密に等しい' === '===は、厳密に等しい');// true or false?

console.log('!== は、厳密に等しくない(3 !== 3.0001)');
console.log(3 !== 3.0001);// true or false?

console.log('>は、より大きい(5 > 4)');
console.log(5 > 4);// true or false?

console.log('>=は、以上(5 * 2  >= 5.0 + 5.0)');
console.log(5 * 2  >= 5.0 + 5.0);// true or false?

console.log('<=は、より小さい((5 <= 6) === (4 <= 4))');
console.log((5 <= 6) === (4 <= 4));// true or false?

console.log('<は、未満(5 < 5)');
console.log(5 > 4);// true or false?

// 論理演算子の効果を体験してみましょう。
console.log('&&は、論理積(AND)。(true && true)');
console.log(true && true);// true or false?

console.log('||は、論理和(OR)。(3 === 3 || true === false)');
console.log(3 === 3 || true === false);// true or false?

console.log('!は、論理否定(NOT)。(!(true || false))');
console.log(!(true || false));// true or false?
