Lesson 12 : Object中心に...

Lesson 12 : Object中心に世界は回っている（おおげさ）

ここからはJavaScriptプログラミングに一歩踏み入って学んでいきましょう。

前回までは、データの表現を中心に学んできました。

また、何かの処理を記述し、定義する場合は関数を使うと便利でした。

さて、いきなりですが、みんなの大好きな猫表現する場合、どのようにしましょう。

var cat_voice_type = ['にゃー', 'シャーッ', 'にゃ？'];
var cat_action_log = {
  heal: 0,
  anger: 0,
  nothing: 0
};
var cat_action = function(act) {
  if (act === '癒やす') {
    cat_action_log.heal++;
    console.log(cat_voice_type[0]);
  } else if (act === '怒る') {
    cat_action_log.anger++;
    console.log(cat_voice_type[1]);
  } else {
    cat_action_log.nothing++;
    console.log(cat_voice_type[2]);
  }
};

鳴き声のタイプが３つあって、人を癒やすときや怒っている時の行動があって、それぞれの行動履歴が記録される。

というものを作ってみました。

うん、じゃあ上のようなコーディングでいいじゃないか、と思いますよね？

でも、これって一つにまとめたほうがより明確に表現できませんかね？

var cat = {
  voice_type: ['にゃー', 'シャーッ', 'にゃ？'],
  action: function(act) {
    if (act === '癒やす') {
      this.action_log.heal++;
      console.log(this.voice_type[0]);
      console.log('癒やし歴:' + this.action_log.heal + '回');
    } else if (act === '怒る') {
      this.action_log.anger++;
      console.log(this.voice_type[1]);
      console.log('怒り歴:' + this.action_log.anger + '回');
    } else {
      this.action_log.nothing++;
      console.log(this.voice_type[2]);
      console.log('無反応歴:' + this.action_log.nothing + '回');
    }
  },
  action_log: {
    heal: 0,
    anger: 0,
    nothing: 0
  }
};

this ってなんでしょう？

this は、呼び出される場所におけるその時のオブジェクトを参照してくれるようです。
