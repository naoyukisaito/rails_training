Lesson 09 : 配列とループ

Lesson 09 : 配列とループ

データの格納方法として便利なものに、配列というものがあります。

これまで、データ（数値や文字列など）を格納するときは１つずつ変数に格納してきました。

しかし、扱うデータによってはまとめて管理、操作したほうが都合の良い場合が多々あります。

学級名簿を考えてみましょう。今までの方法であれば、以下の様な方法で表現したかもしれません。

var student_name01 = 'Masahiro Nakai';
var student_name02 = 'Goro Inagaki';
var student_name03 = 'Takuya Kimura';
var student_name04 = 'Shingo Katori';
var student_name05 = 'Tsuyoshi Kusanagi';

数字が出席番号という感じでしょうか。それにしても5人のクラスなんてよっぽど過疎化のすすんだ所の学校ですね。 さて、冗談はさておき、一般的なクラスの人数というと30人とかでしょうか。この調子で30人分書き続けるってのはスマートではないですね。 配列の仕組みを使うと、以下のように書くことが出来て便利です。

var students_name = ['Masahiro Nakai', 'Goro Inagaki', 'Takuya Kimura', 'Shingo Katori', 'Tsuyoshi Kusanagi'];
var students_number = [1, 2, 3, 4, 5];

javascriptの配列では、データ型をミックスして格納することが出来ます。たとえば、以下の様な感じです。

var students_name_and_number = ['Masahiro Nakai', 1, 'Goro Inagaki', 2, 'Takuya Kimura', 3, 'Shingo Katori', 4, 'Tsuyoshi Kusanagi', 5];

これまでに学んだ変数は、自由に書き換えができました。もちろん、配列も書き換え可能です。

students_number = [3, 2, 5, 4, 1];
students_number = [1];
students_number = ['1番', '2番', '3番', '4番', '5番'];
students_number = '1番, 2番, 3番, 4番, 5番';// javascriptは、元の型を無視して違う型を代入できるので、もちろんこのようなことも可能です(ただし、プログラムの可読性がなくなるので注意しましょう)

もし、出席番号の3番と1番を入れ替えたい時など、一つ一つの要素の順番を書き換えたい時、毎回直すのは面倒ですね。 各要素にアクセスするときは、配列名[要素の番号]という風に書きます。お気づきのように、最初の要素は0番目です。（0番目から数えます）

students_number[0] = 1;
students_number[4] = 3;

要素数（配列の長さ）を取得したい場合は、変数.lengthで取得可能です。

練習問題

var student_name01 = 'Masahiro Nakai';
var student_name02 = 'Goro Inagaki';
var student_name03 = 'Takuya Kimura';
var student_name04 = 'Shingo Katori';
var student_name05 = 'Tsuyoshi Kusanagi';

var students_name = ['Masahiro Nakai', 'Goro Inagaki', 'Takuya Kimura', 'Shingo Katori', 'Tsuyoshi Kusanagi'];
var students_number = [1, 2, 3, 4, 5];

var students_name_and_number = ['Masahiro Nakai', 1, 'Goro Inagaki', 2, 'Takuya Kimura', 3, 'Shingo Katori', 4, 'Tsuyoshi Kusanagi', 5];

students_number = [3, 2, 5, 4, 1];
students_number[0] = 1;
students_number[4] = 3;
console.log(students_number);// 確認してみましょう
console.log(students_number.length);// 要素数（配列の長さ）を取得したい場合は、変数.lengthで取得可能です。確認してみましょう。

// 通常、配列は統一された型の値で表現されます。
// プログラミングをする上で、意味のある整った形で記述することは人にもコンピュータにも優しく、プログラムを構築しやすくします。
// 意味のある内容を型や順番に規則性を持たせることで、前回学んだループと配列は強力な連携をみせてくれます。
// その様子を見てみましょう。
function callOutStudents() {
  for(i = 0; i < students_name_and_number.length / 2; i++) {
    console.log('出席番号' + students_name_and_number[ i * 2 + 1 ] + '番, ' + students_name_and_number[ i * 2 ] + '君！');
    if(i !== 2) {
      console.log('はい！');
    }
    else {
      console.log('ﾁｮﾏﾃﾖ！');
    }
  }
}
callOutStudents();

// 上記では、students_name_and_numberを使って出席番号と名前を呼んでみました。
// 以下で、students_nameとstudents_number両方を使って書き換えてみましょう。余裕があれば、出力内容も自由に変えてみましょう。
// hint : 新しい関数を定義してみましょう。2箇所を書き換えるだけで最低限は実現可能です。
